# Papirothèque Landing Page
This is the landing page for my web! :)

<img width="1835" height="886" alt="image" src="https://github.com/user-attachments/assets/0d51f827-3b1b-4e8d-9dd4-217df53ffab5" />

It features an SVG-based typography animation and serves as a navigational hub between my professional portfolio and GitHub repositories.

## Features
- **SVG Animation:** Custom typography (Portcullion) animation using CSS transitions and hover states.
- **Interactive Navigation:** Integrated links to external platforms (Portfolio and GitHub).
- **Minimalist Design:** Focus on clean aesthetics and fluid movement.

## Tech Stack
- **HTML5:** Page structure.
- **CSS3:** Custom animations, layout, and interactive states.
- **SVG:** Scalable vector graphics for flexible typography rendering.
